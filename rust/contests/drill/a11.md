# A-11: 文字列の比較

配点: 100点

### 問題文

英小文字のみからなる2つの文字列 $S$、$T$ が与えられる。  
$S$ と $T$ が **完全に等しい** なら `Yes`、そうでなければ `No` を出力せよ。

### 制約

- $1 \le |S|, |T| \le 100$
- $S$, $T$ はいずれも英小文字のみからなる

### 入力

```
S
T
```

### 出力

`Yes` または `No` を出力せよ。

---

### 入力例1

```
hello
hello
```

### 出力例1

```
Yes
```

---

### 入力例2

```
hello
world
```

### 出力例2

```
No
```

---

### 入力例3

```
abc
abcd
```

### 出力例3

```
No
```

長さが違うので No。

---

### ヒント

練習ポイント: `String` 同士は `==` で比較できる

```rust
input! {
    s: String,
    t: String,
}
println!("{}", if s == t { "Yes" } else { "No" });
```

C 言語と違い、Rust では `==` が中身を比較してくれる（`strcmp` 相当を自動でやる）。
