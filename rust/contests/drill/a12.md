# A-12: 文字列の連結

配点: 100点

### 問題文

英小文字のみからなる2つの文字列 $S$、$T$ が与えられる。  
$S$ と $T$ をこの順に連結し、**間に1個の空白を入れて** 出力せよ。

### 制約

- $1 \le |S|, |T| \le 100$
- $S$, $T$ はいずれも英小文字のみからなる

### 入力

```
S
T
```

### 出力

`S` 空白 `T` を出力せよ。

---

### 入力例1

```
hello
world
```

### 出力例1

```
hello world
```

---

### 入力例2

```
at
coder
```

### 出力例2

```
at coder
```

---

### 入力例3

```
a
b
```

### 出力例3

```
a b
```

---

### ヒント

練習ポイント: 文字列を連結して出力する。`println!` だけで足りる場合もあれば、いったん変数に組み立てたいこともある。

```rust
// 方法1: そのまま println!
println!("{} {}", s, t);

// 方法2: format! で文字列を作る
let ans = format!("{} {}", s, t);
println!("{}", ans);

// 方法3: + 演算子（左辺は String、右辺は &str）
let ans = s + " " + &t; // s を消費する
```

`format!` は `println!` と同じ書式で **文字列を返す** マクロ。`String` を組み立てたいときの定番。
