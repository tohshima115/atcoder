# A-13: 文字を数値に変換

配点: 100点

### 問題文

数字（`0`〜`9`）のみからなる文字列 $S$ が与えられる。  
各文字を1桁の数字とみなして、それらの **合計** を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は `0`〜`9` のみからなる

### 入力

```
S
```

### 出力

各文字を数値に変換したものの合計を出力せよ。

---

### 入力例1

```
123
```

### 出力例1

```
6
```

$1 + 2 + 3 = 6$。

---

### 入力例2

```
99999
```

### 出力例2

```
45
```

$9 \times 5 = 45$。

---

### 入力例3

```
0
```

### 出力例3

```
0
```

---

### ヒント

練習ポイント: **`char`（文字）を数値に変換する**。これがすごく頻出。3通りの方法を覚えておく。

```rust
// 方法1: char::to_digit(基数)。Option を返すので unwrap
let n: u32 = c.to_digit(10).unwrap();

// 方法2: ASCII の差を使う
let n: i64 = c as i64 - '0' as i64;

// 方法3: いったん文字列にして parse
let n: i64 = c.to_string().parse().unwrap();
```

合計を取るループ全体：

```rust
let mut sum: i64 = 0;
for c in s.chars() {
    sum += (c as i64) - ('0' as i64);
}
println!("{}", sum);
```

イテレータ版：

```rust
let sum: i64 = s.chars().map(|c| (c as i64) - ('0' as i64)).sum();
```

`sum()` を使うときは戻り値の型注釈（ここでは `i64`）が必要なことが多い。
