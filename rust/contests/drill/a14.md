# A-14: 同じ文字の数

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ と、英小文字 $C$（1文字）が与えられる。  
$S$ に文字 $C$ が何個含まれるかを出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみ、$C$ は英小文字1文字

### 入力

```
S
C
```

### 出力

$S$ における $C$ の個数を出力せよ。

---

### 入力例1

```
banana
a
```

### 出力例1

```
3
```

---

### 入力例2

```
mississippi
s
```

### 出力例2

```
4
```

---

### 入力例3

```
abc
z
```

### 出力例3

```
0
```

---

### ヒント

練習ポイント: **入力で `char` を1文字読む** + フィルタ

```rust
input! {
    s: String,
    c: char,   // proconio は char も読める
}
let count = s.chars().filter(|&x| x == c).count();
println!("{}", count);
```

A-9 と似ているが、こちらは「数えたい文字も入力から来る」のがポイント。
`c: char` のように proconio に1文字読ませると、入力の1文字（空白で区切られた1トークンの先頭）が `char` として読まれる。
