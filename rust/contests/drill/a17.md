# A-17: 文字の種類判定

配点: 100点

### 問題文

文字 $C$（英小文字または1桁の数字）が1つ与えられる。  
以下のルールで分類名を出力せよ。

| 種類 | 出力 |
|---|---|
| 母音（a, e, i, o, u） | `vowel` |
| 子音（それ以外の英小文字） | `consonant` |
| 数字（0〜9） | `digit` |

### 制約

- $C$ は英小文字または `0`〜`9` の数字1文字

### 入力

```
C
```

### 出力

分類名を出力せよ。

---

### 入力例1

```
a
```

### 出力例1

```
vowel
```

---

### 入力例2

```
b
```

### 出力例2

```
consonant
```

---

### 入力例3

```
7
```

### 出力例3

```
digit
```

---

### 入力例4

```
u
```

### 出力例4

```
vowel
```

---

### ヒント

練習ポイント: **`match c` で `char` のパターンマッチ**。`|` で複数パターンを並べる。

```rust
input! {
    c: char,
}
let ans = match c {
    'a' | 'e' | 'i' | 'o' | 'u' => "vowel",
    'a'..='z' => "consonant",      // 上のパターンに当たらない英小文字
    '0'..='9' => "digit",
    _ => unreachable!(),
};
println!("{}", ans);
```

ポイント:
- `match` は **上から順に** 評価し、最初に一致したアームを使う
- だから `'a' | 'e' | ...` を先に書けば、その後の `'a'..='z'` で母音が拾われずに済む
- `'a'..='z'` は文字の範囲パターン
