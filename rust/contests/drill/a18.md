# A-18: じゃんけんの勝者

配点: 100点

### 問題文

高橋君と青木君がじゃんけんをした。  
それぞれの手 $A$、$B$ が文字列で与えられる（`rock`, `scissors`, `paper` のいずれか）。

- 高橋君（$A$）が勝つなら `Takahashi`
- 青木君（$B$）が勝つなら `Aoki`
- あいこなら `Draw`

を出力せよ。

### 制約

- $A$, $B$ は `rock`, `scissors`, `paper` のいずれか

### 入力

```
A
B
```

### 出力

`Takahashi`, `Aoki`, `Draw` のいずれかを出力せよ。

---

### 入力例1

```
rock
scissors
```

### 出力例1

```
Takahashi
```

グー vs チョキ → 高橋君（グー）の勝ち。

---

### 入力例2

```
paper
scissors
```

### 出力例2

```
Aoki
```

パー vs チョキ → 青木君（チョキ）の勝ち。

---

### 入力例3

```
rock
rock
```

### 出力例3

```
Draw
```

---

### ヒント

練習ポイント: **`match` で2つの値の組（タプル）をパターンマッチ**

```rust
input! {
    a: String,
    b: String,
}
let ans = match (a.as_str(), b.as_str()) {
    (x, y) if x == y => "Draw",
    ("rock", "scissors") | ("scissors", "paper") | ("paper", "rock") => "Takahashi",
    _ => "Aoki",
};
println!("{}", ans);
```

ポイント:
- `String` を `match` するときは `a.as_str()` で `&str` に変換すると、`"rock"` などの文字列リテラルとパターンマッチできる
- `(x, y) if x == y` は **マッチガード** 付きパターン（条件を追加できる）
- 同じ手 → Draw、特定の組 → Takahashi、それ以外 → Aoki、と整理できる
