# A-20: 隣接要素の差の最大

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ が与えられる。  
隣り合う要素の差 $|A_{i+1} - A_i|$（$i = 1, 2, \dots, N-1$）の **最大値** を出力せよ。  
ただし $N=1$ のときは出力は $0$ とする。

### 制約

- $1 \le N \le 100$
- $-1000 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

隣接差の最大値を整数で出力せよ。

---

### 入力例1

```
5
3 8 1 6 4
```

### 出力例1

```
7
```

差は $|8-3|=5, |1-8|=7, |6-1|=5, |4-6|=2$。最大は7。

---

### 入力例2

```
1
100
```

### 出力例2

```
0
```

要素が1つしかなく差が定義できない → 0。

---

### 入力例3

```
2
-50 50
```

### 出力例3

```
100
```

---

### ヒント

練習ポイント: **ループの範囲 `0..n-1`** と **`n=1` のエッジケース**

```rust
input! {
    n: usize,
    a: [i64; n],
}
let mut ans: i64 = 0;
for i in 0..n - 1 {           // i は 0..=n-2 を動く
    let diff = (a[i + 1] - a[i]).abs();
    if diff > ans {
        ans = diff;
    }
}
println!("{}", ans);
```

ポイント:
- `0..n - 1` は **n-1 個** の数 `0, 1, ..., n-2` を生成する
- `n=1` のとき、`n - 1 == 0` で `0..0` は **空のレンジ**。ループは1回も回らず `ans = 0` のまま → 正しい
- もし `n=0` の制約があったら `n - 1` で `usize` がアンダーフロー（panic）する。今回は `n >= 1` なので大丈夫
- イテレータ版: `let ans = (0..n - 1).map(|i| (a[i + 1] - a[i]).abs()).max().unwrap_or(0);`
