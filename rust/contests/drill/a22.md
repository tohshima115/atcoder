# A-22: 整数の桁数

配点: 100点

### 問題文

非負整数 $N$ が与えられる。  
$N$ を10進数で表したときの桁数を出力せよ。  
ただし $N = 0$ の桁数は $1$ とする。

### 制約

- $0 \le N \le 10^{18}$

### 入力

```
N
```

### 出力

桁数を整数で出力せよ。

---

### 入力例1

```
123
```

### 出力例1

```
3
```

---

### 入力例2

```
0
```

### 出力例2

```
1
```

---

### 入力例3

```
9999999999
```

### 出力例3

```
10
```

---

### ヒント

練習ポイント: **整数の桁を「10で割りながら」見ていく** ループ

```rust
input! {
    n: u64,
}
if n == 0 {
    println!("1");
    return;
}
let mut x = n;
let mut count = 0;
while x > 0 {
    x /= 10;
    count += 1;
}
println!("{}", count);
```

ポイント:
- `x /= 10` で1桁ずつ落としていく
- `N = 0` のときはループが回らず `count = 0` になる → エッジケースとして別扱い
- 別解: `n.to_string().len()` — 文字列にして長さを取ればワンライナー（楽だが、整数の桁操作の感覚は身につかない）
- 制約が $10^{18}$ なので `i64` でも `u64` でも収まる。`u64` なら負の心配がない
