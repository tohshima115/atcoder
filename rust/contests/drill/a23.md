# A-23: 各桁の和

配点: 100点

### 問題文

非負整数 $N$ が与えられる。  
$N$ を10進数で表したときの **各桁の和** を出力せよ。

### 制約

- $0 \le N \le 10^{18}$

### 入力

```
N
```

### 出力

桁の和を整数で出力せよ。

---

### 入力例1

```
123
```

### 出力例1

```
6
```

$1 + 2 + 3 = 6$。

---

### 入力例2

```
0
```

### 出力例2

```
0
```

---

### 入力例3

```
9999999999
```

### 出力例3

```
90
```

$9 \times 10 = 90$。

---

### ヒント

練習ポイント: **`10` で割った余りで1桁取り出す**

```rust
input! {
    n: u64,
}
let mut x = n;
let mut sum: u64 = 0;
while x > 0 {
    sum += x % 10;
    x /= 10;
}
println!("{}", sum);
```

ポイント:
- `x % 10` で **1の位** が取れる
- `x /= 10` で1桁落とす
- この2つをセットで覚える（桁操作の鉄則）
- 別解（文字列経由）: `n.to_string().chars().map(|c| (c as u64) - ('0' as u64)).sum::<u64>()`
- A-13 でも文字→数値の変換をやっている。整数版と文字列版の両方の感覚を持っておくと強い
