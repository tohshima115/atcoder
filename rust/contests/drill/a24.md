# A-24: 各桁の最大値

配点: 100点

### 問題文

正整数 $N$ が与えられる。  
$N$ を10進数で表したときの **各桁のうち最大の値** を出力せよ。

### 制約

- $1 \le N \le 10^{18}$

### 入力

```
N
```

### 出力

最大の桁を整数で出力せよ。

---

### 入力例1

```
9123
```

### 出力例1

```
9
```

桁は 9, 1, 2, 3。最大は9。

---

### 入力例2

```
1111
```

### 出力例2

```
1
```

---

### 入力例3

```
56789
```

### 出力例3

```
9
```

---

### ヒント

練習ポイント: **桁取り出し + 最大値の更新**

```rust
input! {
    n: u64,
}
let mut x = n;
let mut best: u64 = 0;
while x > 0 {
    let d = x % 10;
    if d > best {
        best = d;
    }
    x /= 10;
}
println!("{}", best);
```

ポイント:
- 制約で $N \ge 1$ なので、必ず最低1桁ある → `best = 0` から更新していけば最終的に正しい値になる
- もし $N = 0$ の入力があれば、ループが回らず `best = 0` → これは仕様としては「0という桁」と解釈できる
- 別解: `n.to_string().chars().map(|c| c.to_digit(10).unwrap()).max().unwrap()`
