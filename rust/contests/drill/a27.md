# A-27: 三角形の面積

配点: 100点

### 問題文

底辺 $B$、高さ $H$ の三角形がある。面積を出力せよ。  
出力は **小数第1位まで** とする。

### 制約

- $1 \le B, H \le 1000$
- $B, H$ は整数

### 入力

```
B H
```

### 出力

面積を小数第1位まで出力せよ。

---

### 入力例1

```
4 6
```

### 出力例1

```
12.0
```

面積 = $4 \times 6 / 2 = 12$ → `12.0`。

---

### 入力例2

```
5 3
```

### 出力例2

```
7.5
```

$5 \times 3 / 2 = 7.5$。

---

### 入力例3

```
1 1
```

### 出力例3

```
0.5
```

---

### ヒント

練習ポイント: **整数の `/2` で小数が出るとき**

```rust
input! {
    b: i64,
    h: i64,
}
let area = b as f64 * h as f64 / 2.0;
println!("{:.1}", area);
```

ポイント:
- 整数同士 `b * h / 2` だと小数部分が **切り捨て** られる（`5 * 3 / 2 = 7`）
- `f64` に変換して割れば 7.5 が出る
- `{:.1}` で小数第1位までの出力（必要なら丸めと0埋め）
- 別解: `(b * h) as f64 / 2.0` でも同じ。途中の `b * h` がオーバーフローしないか必ず確認する（今回は最大 1000×1000 で安全）
