# A-28: 中央値

配点: 100点

### 問題文

奇数個の整数 $A_1, A_2, \dots, A_N$ が与えられる。  
昇順に並べたときの **真ん中の値（中央値）** を出力せよ。

### 制約

- $1 \le N \le 99$
- $N$ は奇数
- $-1000 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

中央値を整数で出力せよ。

---

### 入力例1

```
5
3 1 4 1 5
```

### 出力例1

```
3
```

昇順: `1 1 3 4 5` → 真ん中（3番目）は3。

---

### 入力例2

```
1
42
```

### 出力例2

```
42
```

要素が1つだけ → そのまま中央値。

---

### 入力例3

```
7
9 -2 5 5 0 -1 3
```

### 出力例3

```
3
```

昇順: `-2 -1 0 3 5 5 9` → 真ん中（4番目）は3。

---

### ヒント

練習ポイント: **ソート + 真ん中のインデックス**

```rust
input! {
    n: usize,
    mut a: [i64; n],   // mut を付けるとソートできる
}
a.sort();
println!("{}", a[n / 2]);
```

ポイント:
- `a.sort()` で昇順ソート（**配列を変更する** ので `mut` が必要）
- `n` が奇数のとき、真ん中の **0-indexed のインデックス** は `n / 2`
  - n=1 → インデックス0（要素は1個）
  - n=5 → インデックス2（3番目）
  - n=7 → インデックス3（4番目）
- 偶数個の場合は「中央2つの平均」になることが多いが、今回は奇数限定なので簡単

降順ソートしたいときは `a.sort_by(|x, y| y.cmp(x))`（B-3 参照）。
