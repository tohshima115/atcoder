# A-31: 偶数の個数

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ が与えられる。  
**偶数** である要素の個数を出力せよ。

### 制約

- $1 \le N \le 100$
- $-1000 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

偶数の個数を整数で出力せよ。

---

### 入力例1

```
5
1 2 3 4 5
```

### 出力例1

```
2
```

偶数は `2, 4` の2個。

---

### 入力例2

```
3
0 -2 -4
```

### 出力例2

```
3
```

`0` も偶数。負の数の偶数判定もOK。

---

### 入力例3

```
4
1 3 5 7
```

### 出力例3

```
0
```

---

### ヒント

練習ポイント: **偶数判定 `x % 2 == 0`**

```rust
input! {
    n: usize,
    a: [i64; n],
}
let cnt = a.iter().filter(|&&x| x % 2 == 0).count();
println!("{}", cnt);
```

ポイント:
- `%` は剰余演算子。`x % 2` で2で割った余り
- **負の数の剰余** に注意:
  - Rust では `-3 % 2 == -1`（C と同じ。Python とは違う）
  - だから「奇数判定」は `x % 2 != 0` のほうが安全（`== 1` だと負の奇数で外れる）
  - 「偶数判定」は `x % 2 == 0` で OK（0 は符号なし）

豆知識: ビット演算 `x & 1 == 0` でも偶数判定できる（`i64` でも動く）。
