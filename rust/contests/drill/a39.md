# A-39: 文字列の一部を置き換える

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ と、整数 $K$、英小文字 $C$ が与えられる。  
$S$ の **$K$ 文字目** を文字 $C$ に置き換えた文字列を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $1 \le K \le |S|$
- $S$, $C$ は英小文字
- 入力は全て整数または文字列

### 入力

```
S
K C
```

### 出力

置き換え後の文字列を出力せよ。

---

### 入力例1

```
abcde
3 x
```

### 出力例1

```
abxde
```

$3$ 文字目 `c` を `x` に → `abxde`。

---

### 入力例2

```
hello
1 j
```

### 出力例2

```
jello
```

---

### 入力例3

```
a
1 z
```

### 出力例3

```
z
```

---

### ヒント

練習ポイント: **`String` は直接インデックスで書き換えられない** → `Vec<char>` を経由する

- Rust の `String` は `s[2] = 'x'` のような書き換えができない（UTF-8 のため）
- いったん文字の配列にしてから書き換える：
  ```rust
  let mut chars: Vec<char> = s.chars().collect();
  chars[k - 1] = c;
  let ans: String = chars.iter().collect();
  ```
- `K` は 1-indexed なので `k-1`
- `c: char` で1文字を受け取れる（proconio の `input!` で `c: char`）
