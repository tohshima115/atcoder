# A-44: 大小比較

配点: 100点

### 問題文

2つの整数 $A, B$ が与えられる。

- $A > B$ ならば `Greater`
- $A < B$ ならば `Less`
- $A = B$ ならば `Equal`

を出力せよ。

### 制約

- $-100 \le A, B \le 100$
- 入力は整数

### 入力

```
A B
```

### 出力

`Greater` / `Less` / `Equal` のいずれかを出力せよ。

---

### 入力例1

```
3 5
```

### 出力例1

```
Less
```

---

### 入力例2

```
7 2
```

### 出力例2

```
Greater
```

---

### 入力例3

```
4 4
```

### 出力例3

```
Equal
```

---

### ヒント

練習ポイント: **3分岐の `if / else if / else`**

```rust
if a > b {
    println!("Greater");
} else if a < b {
    println!("Less");
} else {
    println!("Equal");
}
```

- 条件は上から順に評価され、最初に真になったものだけ実行される
- `else if` で「それ以外かつ別の条件」をつなげる
- 別解: `a.cmp(&b)` は `Ordering`（`Greater`/`Less`/`Equal`）を返すので `match` で書ける
