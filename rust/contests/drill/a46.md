# A-46: 部分文字列の取り出し

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ と、整数 $L, R$ が与えられる。  
$S$ の **$L$ 文字目から $R$ 文字目まで**（両端を含む）を取り出して出力せよ。

### 制約

- $1 \le L \le R \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
L R
```

### 出力

部分文字列を出力せよ。

---

### 入力例1

```
abcde
2 4
```

### 出力例1

```
bcd
```

$2$ 文字目から $4$ 文字目 → `bcd`。

---

### 入力例2

```
hello
1 5
```

### 出力例2

```
hello
```

全体を取り出す。

---

### 入力例3

```
world
3 3
```

### 出力例3

```
r
```

1文字だけ取り出す。

---

### ヒント

練習ポイント: **文字列のスライス** と **1-indexed → 0-indexed**

- 英小文字（1バイト文字）だけなら、バイトスライスで取り出せる：
  ```rust
  let ans = &s[l-1..r];   // L文字目(l-1) から R文字目(r, 含む手前まで)
  ```
- `L` は 1-indexed なので `l-1`。`R` 文字目を含めたいので終端は `r`（`..r` は `r-1` 番目まで＝ `R` 文字目まで）
- 注意：日本語など複数バイト文字が混じると `&s[..]` はパニックする。その場合は `Vec<char>` を経由する：
  ```rust
  let chars: Vec<char> = s.chars().collect();
  let ans: String = chars[l-1..r].iter().collect();
  ```
