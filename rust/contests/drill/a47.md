# A-47: 種類の数

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ が与えられる。  
この中に **何種類の異なる値** があるかを出力せよ。

### 制約

- $1 \le N \le 100$
- $0 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

異なる値の種類数を出力せよ。

---

### 入力例1

```
4
1 2 2 3
```

### 出力例1

```
3
```

`1, 2, 3` の3種類（`2` が2回出ても1種類）。

---

### 入力例2

```
3
5 5 5
```

### 出力例2

```
1
```

---

### 入力例3

```
4
1 2 3 4
```

### 出力例3

```
4
```

---

### ヒント

練習ポイント: **`HashSet` で重複を除く**

- `HashSet` は「同じ値を1つしか持たない集合」。入れるだけで重複が消える：
  ```rust
  use std::collections::HashSet;

  let set: HashSet<i64> = a.iter().cloned().collect();
  println!("{}", set.len());
  ```
- `a.iter().cloned()` で `&i64` → `i64` にして集める（`.copied()` でも同じ）
- `set.len()` が種類数
- B問題以降で「重複を消したい」「既に出たか調べたい」場面で `HashSet` は頻出
