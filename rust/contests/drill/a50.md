# A-50: 隣接する差の最大

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ が与えられる。
隣り合う2要素の差 $A_{i+1} - A_i$ の **最大値** を出力せよ（$1 \le i \le N-1$）。

### 制約

- $2 \le N \le 100$
- $-10^9 \le A_i \le 10^9$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

$A_{i+1} - A_i$ の最大値を出力せよ。

---

### 入力例1

```
5
1 3 2 8 4
```

### 出力例1

```
6
```

差は `3-1=2, 2-3=-1, 8-2=6, 4-8=-4` → 最大は $6$。

---

### 入力例2

```
3
5 3 1
```

### 出力例2

```
-2
```

単調減少なら差は全て負。最大は $-2$。

---

### 入力例3

```
2
7 7
```

### 出力例3

```
0
```

同じ値なら差は $0$。

---

### ヒント

練習ポイント: **`.windows(2)`** で隣接ペアを取り出す

- `.windows(2)` は **長さ2のスライス** を順に返すイテレータ:
  ```rust
  let mx = a.windows(2).map(|w| w[1] - w[0]).max().unwrap();
  ```
- `w[0]` が前、`w[1]` が後ろ
- 別解: インデックスで回す
  ```rust
  let mut mx = i64::MIN;
  for i in 0..n-1 {
      mx = mx.max(a[i+1] - a[i]);
  }
  ```
- `windows` は「隣接3要素」「k連続」を扱う時にも超頻出
