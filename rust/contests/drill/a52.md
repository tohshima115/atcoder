# A-52: 値の最初の出現位置

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ と整数 $X$ が与えられる。
$A$ の中で **最初に $X$ が出現する位置**（1-indexed）を出力せよ。
$X$ が含まれない場合は `-1` を出力せよ。

### 制約

- $1 \le N \le 100$
- $0 \le A_i, X \le 1000$
- 入力は全て整数

### 入力

```
N X
A_1 A_2 ... A_N
```

### 出力

最初に出現する位置（1-indexed）、無ければ `-1`。

---

### 入力例1

```
5 3
1 2 3 4 3
```

### 出力例1

```
3
```

`3` は3番目（と5番目）に現れる。最初は3番目。

---

### 入力例2

```
4 9
1 2 3 4
```

### 出力例2

```
-1
```

含まれない。

---

### 入力例3

```
1 7
7
```

### 出力例3

```
1
```

---

### ヒント

練習ポイント: **`.iter().position()`** と **`Option` の扱い**

- `position` は条件を満たす最初の要素の **インデックス**（0-indexed）を `Option<usize>` で返す:
  ```rust
  let pos = a.iter().position(|&v| v == x);
  ```
- 戻り値が `Option` なので、`match` で分岐:
  ```rust
  match a.iter().position(|&v| v == x) {
      Some(i) => println!("{}", i + 1),   // 1-indexed に直す
      None => println!("-1"),
  }
  ```
- または `map_or`:
  ```rust
  let ans: i64 = a.iter().position(|&v| v == x).map_or(-1, |i| (i + 1) as i64);
  println!("{}", ans);
  ```
- `contains(&x)` は「あるかないか」だけ、`position` は「あるなら位置も」
