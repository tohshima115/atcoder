# A-54: 文字列中の特定文字の個数

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ と、英小文字 $C$ が与えられる。$S$ に含まれる文字 $C$ の **個数** を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$, $C$ は英小文字のみ

### 入力

```
S
C
```

### 出力

$S$ に含まれる $C$ の個数を出力せよ。

---

### 入力例1

```
abracadabra
a
```

### 出力例1

```
5
```

`abracadabra` の中の `a` は5個。

---

### 入力例2

```
hello
z
```

### 出力例2

```
0
```

含まれない。

---

### 入力例3

```
mississippi
s
```

### 出力例3

```
4
```

---

### ヒント

練習ポイント: **`chars().filter().count()`**

- 1行で書ける:
  ```rust
  let cnt = s.chars().filter(|&ch| ch == c).count();
  println!("{}", cnt);
  ```
- ポイント:
  - `chars()` は `char` を **値で返す**（[[drill-a41]] 参照）ので、`filter` のクロージャは `|&ch|` で受ければ `ch: char`
  - `iter()` を使った場合の `|&&x|` とは別パターン
- `count()` は `usize` を返す
