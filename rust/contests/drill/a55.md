# A-55: 2数列の要素ごとの和

配点: 100点

### 問題文

長さ $N$ の2つの数列 $A = (A_1, \dots, A_N)$, $B = (B_1, \dots, B_N)$ が与えられる。
**各要素ごとの和** $C_i = A_i + B_i$ からなる数列を空白区切りで出力せよ。

### 制約

- $1 \le N \le 100$
- $0 \le A_i, B_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
B_1 B_2 ... B_N
```

### 出力

要素ごとの和を空白区切りで1行に出力せよ。

---

### 入力例1

```
4
1 2 3 4
10 20 30 40
```

### 出力例1

```
11 22 33 44
```

---

### 入力例2

```
3
5 5 5
1 2 3
```

### 出力例2

```
6 7 8
```

---

### 入力例3

```
1
100
200
```

### 出力例3

```
300
```

---

### ヒント

練習ポイント: **`.zip()`** で2つのイテレータをペアにする

- `zip` は2つのイテレータを「同じ位置のペア」に束ねる:
  ```rust
  let c: Vec<i64> = a.iter().zip(b.iter()).map(|(x, y)| x + y).collect();
  ```
  - `a.iter().zip(b.iter())` で `(&i64, &i64)` のイテレータ
  - `|(x, y)| x + y` で参照のまま足せる（`+` は `&i64` 同士でも動く）
- 出力は今までと同じパターン:
  ```rust
  let s: Vec<String> = c.iter().map(|x| x.to_string()).collect();
  println!("{}", s.join(" "));
  ```
- `zip` は3つ以上のイテレータを束ねたいときも、`.zip()` を連結すれば書ける（`((x, y), z)` のような形になる）
