# A-8: 部分文字列の取り出し

配点: 100点

### 問題文

英小文字のみからなる長さ5以上の文字列 $S$ が与えられる。  
$S$ の **2文字目から4文字目までの3文字** を出力せよ（1-indexed）。

### 制約

- $5 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

$S$ の 2 文字目〜 4 文字目を出力せよ。

---

### 入力例1

```
hello
```

### 出力例1

```
ell
```

`h e l l o` の 2〜4 文字目（1-indexed）は `e l l`。

---

### 入力例2

```
atcoder
```

### 出力例2

```
tco
```

---

### 入力例3

```
abcde
```

### 出力例3

```
bcd
```

---

### ヒント

練習ポイント: **問題文の1-indexed と Rust の0-indexed の変換**

問題文の「2文字目から4文字目まで」は1-indexedで、Rustのスライスでは：

```rust
let sub = &s[1..4]; // 0-indexed: 添字 1, 2, 3 の3文字
```

Rust のスライス `s[a..b]` は「**a 番目以上 b 番目未満**」（半開区間）。
「**2 文字目から 4 文字目まで（両端含む）**」は 1-indexed で 2, 3, 4 → 0-indexed で 1, 2, 3 → `s[1..4]`。

**注意**: スライスはバイト境界。`&str` の場合は ASCII なら安全だが、UTF-8 のマルチバイト境界に当たるとパニックする。
