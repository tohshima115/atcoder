# A-9: 'a' の個数

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。  
$S$ に含まれる文字 `a` の個数を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

`a` の個数を整数で出力せよ。

---

### 入力例1

```
banana
```

### 出力例1

```
3
```

`b a n a n a` のうち `a` が3つ。

---

### 入力例2

```
xyz
```

### 出力例2

```
0
```

`a` を含まないので 0。

---

### 入力例3

```
aaaa
```

### 出力例3

```
4
```

---

### ヒント

練習ポイント: **特定の文字をカウントする**。2通り示す。

```rust
// 方法1: for + カウンタ（A-4 と同じパターン）
let mut count = 0;
for c in s.chars() {
    if c == 'a' {
        count += 1;
    }
}

// 方法2: イテレータの filter + count（1行）
let count = s.chars().filter(|&c| c == 'a').count();
```

`filter(|&c| c == 'a')` の `&c` は「参照を外して `c` 自体を扱う」ためのパターン。
`count()` は `usize` を返すので、`println!("{}", count)` でそのまま出力できる。
