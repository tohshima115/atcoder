# A-1: 合格判定

配点: 100点

### 問題文

高橋君はテストを2回受けた。1回目は $A$ 点、2回目は $B$ 点だった。  
2回の合計が $N$ 点以上なら `Pass`、未満なら `Fail` を出力せよ。

### 制約

- $1 \le A, B \le 100$
- $1 \le N \le 200$
- 入力は全て整数

### 入力

```
A B N
```

### 出力

`Pass` または `Fail` を出力せよ。

---

### 入力例1

```
60 70 120
```

### 出力例1

```
Pass
```

合計130点 >= 120点なので合格。

---

### 入力例2

```
40 50 100
```

### 出力例2

```
Fail
```

合計90点 < 100点なので不合格。

---

### 入力例3

```
50 50 100
```

### 出力例3

```
Pass
```

ちょうど100点でも合格。

---

### ヒント

練習ポイント: `input!` で複数の値を読む / `if` で文字列を出力するパターン

```rust
println!("{}", if 条件 { "Pass" } else { "Fail" });
```
