# A-10: 大文字に変換

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。  
$S$ をすべて **大文字** に変換して出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

$S$ を大文字に変換した結果を出力せよ。

---

### 入力例1

```
hello
```

### 出力例1

```
HELLO
```

---

### 入力例2

```
atcoder
```

### 出力例2

```
ATCODER
```

---

### 入力例3

```
a
```

### 出力例3

```
A
```

---

### ヒント

練習ポイント: 文字列の大文字化は標準メソッドがある

```rust
let upper = s.to_uppercase();
println!("{}", upper);
```

`to_uppercase()` は新しい `String` を返す（元の `s` は変わらない）。
逆向きは `to_lowercase()`。
