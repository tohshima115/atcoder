# A-15: 月から季節へ

配点: 100点

### 問題文

月を表す整数 $M$（$1 \le M \le 12$）が与えられる。  
以下のルールで季節名を出力せよ。

| 月 | 季節 |
|---|---|
| 3, 4, 5 | `spring` |
| 6, 7, 8 | `summer` |
| 9, 10, 11 | `autumn` |
| 12, 1, 2 | `winter` |

### 制約

- $1 \le M \le 12$

### 入力

```
M
```

### 出力

季節名を出力せよ。

---

### 入力例1

```
4
```

### 出力例1

```
spring
```

---

### 入力例2

```
8
```

### 出力例2

```
summer
```

---

### 入力例3

```
12
```

### 出力例3

```
winter
```

---

### 入力例4

```
1
```

### 出力例4

```
winter
```

---

### ヒント

練習ポイント: **`match` で範囲パターン `a..=b` を使う**

```rust
let ans = match m {
    3..=5 => "spring",
    6..=8 => "summer",
    9..=11 => "autumn",
    _ => "winter",       // それ以外は冬（12, 1, 2）
};
println!("{}", ans);
```

- `3..=5` は **3 以上 5 以下** を意味するパターン（半開ではなく閉区間）
- `_` は「それ以外すべて」のワイルドカード。これがないと `match` が **網羅性エラー** で落ちる
- `match` は **式** なので、値を変数に代入したり `println!` に直接渡したりできる
