# A-16: 数字を英語に

配点: 100点

### 問題文

整数 $N$（$1 \le N \le 5$）が与えられる。  
$N$ に対応する英単語を出力せよ。

| N | 出力 |
|---|---|
| 1 | `one` |
| 2 | `two` |
| 3 | `three` |
| 4 | `four` |
| 5 | `five` |

### 制約

- $1 \le N \le 5$

### 入力

```
N
```

### 出力

対応する英単語を出力せよ。

---

### 入力例1

```
3
```

### 出力例1

```
three
```

---

### 入力例2

```
1
```

### 出力例2

```
one
```

---

### 入力例3

```
5
```

### 出力例3

```
five
```

---

### ヒント

練習ポイント: **`match` でリテラル値のパターン**

```rust
let ans = match n {
    1 => "one",
    2 => "two",
    3 => "three",
    4 => "four",
    5 => "five",
    _ => unreachable!(),   // 制約上ここには来ない
};
println!("{}", ans);
```

- 各 `=>` の右辺は **同じ型** でないといけない（ここでは `&str`）
- `unreachable!()` は「ここには絶対来ない」と Rust に教えるマクロ。万が一来てしまったらパニックする
- 制約で `1..=5` が保証されているので、`_ => ""` でも動くが、`unreachable!()` のほうが意図が伝わる
