# A-19: 最初と最後の差

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ が与えられる。  
最初の要素と最後の要素の **差の絶対値** $|A_1 - A_N|$ を出力せよ。  
ただし $N=1$ のときは出力は $0$ とする。

### 制約

- $1 \le N \le 100$
- $-1000 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

差の絶対値を整数で出力せよ。

---

### 入力例1

```
5
10 3 7 2 5
```

### 出力例1

```
5
```

$|10 - 5| = 5$。

---

### 入力例2

```
1
42
```

### 出力例2

```
0
```

要素が1つしかないので差は0。

---

### 入力例3

```
3
-5 100 -8
```

### 出力例3

```
3
```

$|-5 - (-8)| = 3$。

---

### ヒント

練習ポイント: **`N=1` のエッジケース**。配列の末尾を取ろうとすると添字計算で詰まる。

```rust
input! {
    n: usize,
    a: [i64; n],
}
let ans = if n == 1 { 0 } else { (a[0] - a[n - 1]).abs() };
println!("{}", ans);
```

ポイント:
- 「最後の要素」は `a[n - 1]`（**1-indexed の N 番目 = 0-indexed の N-1 番目**）
- `n=1` なら `a[0] == a[n-1]` で差は0だが、結果として `if` で分岐しなくても答えは0なので、実は `(a[0] - a[n-1]).abs()` だけで両方カバーできる
- 「N=1 でも壊れないか」を常に意識する習慣を作る（A-3 で踏んだ罠と同じ系統）
- `abs()` は絶対値。`i64` のメソッド
