# A-2: バーコードチェック

配点: 100点

### 問題文

商品のバーコードはアルファベット3文字+数字3文字の、全部で6文字の形式である。  
与えられたバーコード $S$ の数字部分が $100$ 以上 $500$ 以下なら `OK`、そうでなければ `NG` を出力せよ。

### 制約

- $S$ は長さ6の文字列
- $S$ の先頭3文字は英小文字
- $S$ の末尾3文字は数字

### 入力

```
S
```

### 出力

`OK` または `NG` を出力せよ。

---

### 入力例1

```
abc250
```

### 出力例1

```
OK
```

数字部分は250。100以上500以下なのでOK。

---

### 入力例2

```
xyz050
```

### 出力例2

```
NG
```

数字部分は50。100未満なのでNG。

---

### 入力例3

```
foo100
```

### 出力例3

```
OK
```

---

### 入力例4

```
bar501
```

### 出力例4

```
NG
```

---

### ヒント

練習ポイント: 文字列の後ろ3文字を数値に変換する

```rust
let num: i64 = s[3..].parse().unwrap();
```
