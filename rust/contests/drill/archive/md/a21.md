# A-21: 最大値を除いた合計

配点: 100点

### 問題文

$N$ 個の整数が与えられる。  
**最大値を1つだけ除いた残り** の合計を出力せよ。  
ただし $N=1$ のときは出力は $0$ とする（残りがないため）。

### 制約

- $1 \le N \le 100$
- $1 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

最大値を1つ除いた残りの合計を出力せよ。

---

### 入力例1

```
5
3 1 4 1 5
```

### 出力例1

```
9
```

最大は5。残り `3 1 4 1` の合計は9。

---

### 入力例2

```
1
100
```

### 出力例2

```
0
```

唯一の要素を除くと空。合計は0。

---

### 入力例3

```
4
7 7 7 7
```

### 出力例3

```
21
```

最大値（7）が複数あっても、**1個だけ** 除く → `7 + 7 + 7 = 21`。

---

### ヒント

練習ポイント: **「全体の合計」から「最大値」を引く** が一番楽

```rust
input! {
    n: usize,
    a: [i64; n],
}
let sum: i64 = a.iter().sum();
let max: i64 = *a.iter().max().unwrap();   // n>=1 で空でないので OK
let ans = sum - max;
println!("{}", ans);
```

ポイント:
- 同じ値が複数あっても `max` は1つの数値なので、引かれるのも1個ぶんだけ → 仕様と一致
- `n=1` のとき `sum == max` で `ans = 0` → 仕様と一致
- `.iter().max()` は `Option<&i64>` を返すので、`unwrap()` で参照を取り、`*` で値に
- `.iter().sum()` は型注釈が必要（`let sum: i64`）
