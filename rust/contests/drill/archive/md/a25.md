# A-25: 西暦から世紀へ

配点: 100点

### 問題文

西暦 $Y$ 年が何世紀かを出力せよ。  
ルール:

- 1年〜100年 → 1世紀
- 101年〜200年 → 2世紀
- 201年〜300年 → 3世紀
- ...

### 制約

- $1 \le Y \le 10^9$
- $Y$ は整数

### 入力

```
Y
```

### 出力

何世紀かを整数で出力せよ。

---

### 入力例1

```
1
```

### 出力例1

```
1
```

---

### 入力例2

```
100
```

### 出力例2

```
1
```

100年も1世紀。

---

### 入力例3

```
2024
```

### 出力例3

```
21
```

2024年は21世紀。

---

### 入力例4

```
2001
```

### 出力例4

```
21
```

2001年は21世紀の最初の年。

---

### ヒント

練習ポイント: **整数演算で「切り上げ」を表現する**

```rust
input! {
    y: i64,
}
let ans = (y + 99) / 100;
println!("{}", ans);
```

ポイント:
- 切り上げ除算の常套句: `(分子 + 分母 - 1) / 分母`
- 100年 → `(100 + 99) / 100 = 199 / 100 = 1`（整数除算なので小数切り捨て）
- 101年 → `(101 + 99) / 100 = 200 / 100 = 2`
- 別解: `(y - 1) / 100 + 1` — こちらでも同じ結果になる
- 浮動小数点 `(y as f64 / 100.0).ceil() as i64` でもいいが、誤差や型変換が増えるので整数演算で済むなら整数で
