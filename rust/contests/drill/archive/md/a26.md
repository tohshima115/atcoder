# A-26: 平均点（小数2桁）

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ が与えられる。  
平均を **小数第2位まで** 出力せよ（小数第3位以下は0でも省略しないこと）。

### 制約

- $1 \le N \le 100$
- $0 \le A_i \le 100$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

平均を小数第2位まで出力せよ。

---

### 入力例1

```
3
60 70 80
```

### 出力例1

```
70.00
```

---

### 入力例2

```
4
1 2 3 4
```

### 出力例2

```
2.50
```

平均は 2.5 だが、小数2桁固定で `2.50` と出力。

---

### 入力例3

```
1
99
```

### 出力例3

```
99.00
```

---

### ヒント

練習ポイント: **小数フォーマット `{:.桁数}`** と **整数→浮動小数の変換**

```rust
input! {
    n: usize,
    a: [i64; n],
}
let sum: i64 = a.iter().sum();
let avg = sum as f64 / n as f64;
println!("{:.2}", avg);
```

ポイント:
- `i64` 同士で割ると整数除算（切り捨て）になるので、平均には `f64` に変換してから割る
- `as f64` で型変換
- `{:.2}` は **小数第2位まで** 出力（必要なら0埋め）。`{:.4}` なら4桁
- AtCoder では「絶対誤差または相対誤差が 10^-6 以下なら正解」というジャッジが多い。**指定があれば桁数を厳守する** が、指定が緩いなら `{:.10}` などにしておけば安全
