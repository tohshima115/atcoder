# A-29: 上位K個の合計

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ と整数 $K$ が与えられる。  
**大きい順に上位 $K$ 個** の合計を出力せよ。

### 制約

- $1 \le K \le N \le 100$
- $0 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N K
A_1 A_2 ... A_N
```

### 出力

上位 $K$ 個の合計を整数で出力せよ。

---

### 入力例1

```
5 3
1 5 2 8 3
```

### 出力例1

```
16
```

降順: `8 5 3 2 1` → 上位3個は `8 5 3` → 合計16。

---

### 入力例2

```
4 4
10 20 30 40
```

### 出力例2

```
100
```

全て選ぶ → 合計100。

---

### 入力例3

```
3 1
0 0 100
```

### 出力例3

```
100
```

---

### ヒント

練習ポイント: **降順ソート + 前から K 個を取る**

```rust
input! {
    n: usize,
    k: usize,
    mut a: [i64; n],
}
a.sort_by(|x, y| y.cmp(x));               // 降順
let sum: i64 = a.iter().take(k).sum();    // 前から k 個の合計
println!("{}", sum);
```

ポイント:
- `sort_by(|x, y| y.cmp(x))` で降順ソート（B-3 と同じパターン）
- `.take(k)` は **イテレータの先頭 k 個** を取り出すアダプタ
- `.sum()` は型注釈が必要なことが多い（`let sum: i64 = ...`）
- 別解: 昇順ソートして後ろから k 個 → `a.iter().rev().take(k).sum()`
