# A-30: K以上の要素の個数

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ と整数 $K$ が与えられる。  
$A_i \ge K$ を満たす要素の **個数** を出力せよ。

### 制約

- $1 \le N \le 100$
- $-1000 \le K, A_i \le 1000$
- 入力は全て整数

### 入力

```
N K
A_1 A_2 ... A_N
```

### 出力

条件を満たす要素の個数を整数で出力せよ。

---

### 入力例1

```
5 50
80 30 90 40 50
```

### 出力例1

```
3
```

50以上は `80, 90, 50` の3個。

---

### 入力例2

```
4 100
10 20 30 40
```

### 出力例2

```
0
```

---

### 入力例3

```
3 -5
-10 0 5
```

### 出力例3

```
2
```

`-5以上` は `0, 5` の2個。`-10` は条件不成立。

---

### ヒント

練習ポイント: **`filter` で条件を満たすものだけ通し、`count` で個数を取る**

```rust
input! {
    n: usize,
    k: i64,
    a: [i64; n],
}
let cnt = a.iter().filter(|&&x| x >= k).count();
println!("{}", cnt);
```

ポイント:
- `a.iter()` は `&i64` を返す。`filter` のクロージャは `&&i64` を受け取る形になる
- `|&&x| x >= k` で「2回参照を外して `x: i64` として扱う」
- `count()` は `usize` を返す。`println!("{}", cnt)` でそのまま出る

A-9 の特定文字カウントと同じ「filter + count」のパターン。文字でも数値でも同じ書き方。
