# A-32: 3でも5でも割れない数

配点: 100点

### 問題文

整数 $N$ が与えられる。  
$1$ 以上 $N$ 以下の整数のうち、$3$ の倍数でも $5$ の倍数でもないものの個数を出力せよ。

### 制約

- $1 \le N \le 10^6$

### 入力

```
N
```

### 出力

条件を満たす整数の個数を出力せよ。

---

### 入力例1

```
15
```

### 出力例1

```
8
```

1〜15 のうち、3 の倍数でも 5 の倍数でもないのは `1, 2, 4, 7, 8, 11, 13, 14` の 8 個。

---

### 入力例2

```
1
```

### 出力例2

```
1
```

`1` は3でも5でも割れない → 1個。

---

### 入力例3

```
30
```

### 出力例3

```
16
```

---

### ヒント

練習ポイント: **for ループ + 倍数判定（FizzBuzz の親戚）**

```rust
input! {
    n: i64,
}
let mut cnt = 0;
for i in 1..=n {
    if i % 3 != 0 && i % 5 != 0 {
        cnt += 1;
    }
}
println!("{}", cnt);
```

ポイント:
- `1..=n` は **1 から n まで（両端含む）** のレンジ。`..` ではなく `..=`
- `i % 3 != 0 && i % 5 != 0` で「3でも5でも割り切れない」
- `N=10^6` でも余裕で間に合う（ループ1回が定数時間）
- イテレータ版: `(1..=n).filter(|i| i % 3 != 0 && i % 5 != 0).count()`

数学的な公式（包除原理）でも解けるが、最初は素直にループでよい。
