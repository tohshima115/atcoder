# A-33: 距離

配点: 100点

### 問題文

数直線上の2点 $A$、$B$ が与えられる。  
2点間の距離 $|A - B|$ を出力せよ。

### 制約

- $-10^9 \le A, B \le 10^9$
- $A, B$ は整数

### 入力

```
A B
```

### 出力

距離を整数で出力せよ。

---

### 入力例1

```
3 10
```

### 出力例1

```
7
```

---

### 入力例2

```
-5 5
```

### 出力例2

```
10
```

---

### 入力例3

```
-100 -30
```

### 出力例3

```
70
```

---

### ヒント

練習ポイント: **`abs()` の使い方** と **型の選択**

```rust
input! {
    a: i64,
    b: i64,
}
let ans = (a - b).abs();
println!("{}", ans);
```

ポイント:
- `i64` には `abs()` メソッドが直接生えている
- 制約が $\pm 10^9$ なので `i32` でもギリギリ収まる（`i32::MAX` は約 2.1×10^9）が、念のため `i64` を使う癖をつける
- `a - b` が `i64::MIN` の場合は `abs()` でオーバーフロー（パニック）するが、今回の制約では起きない

**罠**: もし `a, b` を `u64`（符号なし）で読んでしまうと `a - b` で負の値が表現できずパニックする。**差を取る可能性のある値は `i64` で読む** のが安全。
