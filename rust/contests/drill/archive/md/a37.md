# A-37: 2つの要素を交換

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ と、$2$ つの位置 $P, Q$ が与えられる。  
$A$ の $P$ 番目と $Q$ 番目の要素を **交換した** 数列を空白区切りで出力せよ。

### 制約

- $1 \le N \le 100$
- $1 \le P, Q \le N$
- $0 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N P Q
A_1 A_2 ... A_N
```

### 出力

交換後の数列を空白区切りで1行に出力せよ。

---

### 入力例1

```
5 2 4
1 2 3 4 5
```

### 出力例1

```
1 4 3 2 5
```

$2$ 番目（$2$）と $4$ 番目（$4$）を交換 → $(1, 4, 3, 2, 5)$。

---

### 入力例2

```
3 1 3
10 20 30
```

### 出力例2

```
30 20 10
```

---

### 入力例3

```
2 1 2
5 5
```

### 出力例3

```
5 5
```

同じ値でも交換処理は行う（結果は変わらない）。

---

### ヒント

練習ポイント: **`.swap()`** と **1-indexed → 0-indexed**

- `Vec` には2つの要素を入れ替えるメソッドがある：
  ```rust
  a.swap(p - 1, q - 1);   // P番目とQ番目を交換（mut が必要）
  ```
- `P`, `Q` は 1-indexed なので、配列では `p-1`, `q-1`
- 自分で `let tmp = a[i]; a[i] = a[j]; a[j] = tmp;` と書いてもよいが、`.swap()` が一発
