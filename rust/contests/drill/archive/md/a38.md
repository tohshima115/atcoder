# A-38: 数列への挿入

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ と、整数 $K, X$ が与えられる。  
$A$ の **前から $K$ 番目になる位置に** 値 $X$ を挿入した数列を空白区切りで出力せよ。  
（挿入後、$X$ が $K$ 番目の要素になる。元の $K$ 番目以降は1つずつ後ろにずれる）

### 制約

- $1 \le N \le 100$
- $1 \le K \le N + 1$
- $0 \le A_i, X \le 1000$
- 入力は全て整数

### 入力

```
N K X
A_1 A_2 ... A_N
```

### 出力

挿入後の数列（長さ $N+1$）を空白区切りで1行に出力せよ。

---

### 入力例1

```
3 2 9
1 2 3
```

### 出力例1

```
1 9 2 3
```

$2$ 番目に $9$ を挿入 → $(1, 9, 2, 3)$。

---

### 入力例2

```
3 1 5
1 2 3
```

### 出力例2

```
5 1 2 3
```

先頭に挿入。

---

### 入力例3

```
3 4 7
1 2 3
```

### 出力例3

```
1 2 3 7
```

$K = N+1$ のときは末尾に追加。

---

### ヒント

練習ポイント: **`.insert()`** と **`.push()`** の違い

- 指定位置への挿入：
  ```rust
  a.insert(k - 1, x);   // K番目（0-indexedでk-1）に x を挿入（mut が必要）
  ```
- 末尾への追加だけなら `.push()`：
  ```rust
  a.push(x);   // 配列の最後に x を足す
  ```
- `insert(k-1, x)` は「インデックス `k-1` に x が来て、以降が後ろにずれる」。`k-1 == a.len()` なら末尾追加と同じ
