# A-40: 数列を左に回転

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ と整数 $K$ が与えられる。  
$A$ を **左に $K$ 回** 回転させた数列を空白区切りで出力せよ。  
左に1回の回転とは、先頭の要素を取り除いて末尾に付け替える操作である。

### 制約

- $1 \le N \le 100$
- $0 \le K \le N$
- $0 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N K
A_1 A_2 ... A_N
```

### 出力

回転後の数列を空白区切りで1行に出力せよ。

---

### 入力例1

```
5 2
1 2 3 4 5
```

### 出力例1

```
3 4 5 1 2
```

左に2回回転 → 先頭の `1`, `2` が後ろに回る → $(3, 4, 5, 1, 2)$。

---

### 入力例2

```
3 0
10 20 30
```

### 出力例2

```
10 20 30
```

$K = 0$ なら変化なし。

---

### 入力例3

```
4 4
1 2 3 4
```

### 出力例3

```
1 2 3 4
```

$K = N$ なら一周して元に戻る。

---

### ヒント

練習ポイント: **`.rotate_left()`**

- `Vec` には回転メソッドがある：
  ```rust
  a.rotate_left(k);   // 左にk回転（mut が必要）
  ```
- 右回転は `.rotate_right(k)`
- `K` が `N` を超える場合は `a.rotate_left(k % n)` のように余りを取る必要があるが、今回の制約は $K \le N$ なのでそのまま渡せる
- 自分でスライスを組み合わせて作ることもできる：`[a[k..], a[..k]].concat()`
