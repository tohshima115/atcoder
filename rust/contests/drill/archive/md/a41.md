# A-41: 回文判定

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。  
$S$ が **回文**（前から読んでも後ろから読んでも同じ）であれば `Yes`、そうでなければ `No` を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

回文なら `Yes`、そうでなければ `No` を出力せよ。

---

### 入力例1

```
level
```

### 出力例1

```
Yes
```

---

### 入力例2

```
hello
```

### 出力例2

```
No
```

---

### 入力例3

```
a
```

### 出力例3

```
Yes
```

1文字は回文。

---

### ヒント

練習ポイント: **逆順にして元と比較**

- 文字列を反転して元と比べるだけ：
  ```rust
  let rev: String = s.chars().rev().collect();
  if s == rev {
      println!("Yes");
  } else {
      println!("No");
  }
  ```
- `s == rev` で `String` 同士を直接比較できる
- 別解（前後から突き合わせ）: `chars` を `Vec<char>` にして `i` 番目と `len-1-i` 番目を比較するループ
