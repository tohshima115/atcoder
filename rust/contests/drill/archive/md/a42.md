# A-42: 大文字と小文字を入れ替える

配点: 100点

### 問題文

英大文字・英小文字からなる文字列 $S$ が与えられる。  
各文字について、大文字は小文字に、小文字は大文字に **入れ替えた** 文字列を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英大文字・英小文字のみからなる

### 入力

```
S
```

### 出力

大文字小文字を入れ替えた文字列を出力せよ。

---

### 入力例1

```
Hello
```

### 出力例1

```
hELLO
```

---

### 入力例2

```
AtCoder
```

### 出力例2

```
aTcODER
```

---

### 入力例3

```
abc
```

### 出力例3

```
ABC
```

---

### ヒント

練習ポイント: **`char` の大文字小文字判定と変換**

- 1文字 `c: char` について：
  - `c.is_uppercase()` … 大文字か判定
  - `c.to_ascii_uppercase()` / `c.to_ascii_lowercase()` … 変換
- `map` で各文字を変換して `collect`：
  ```rust
  let ans: String = s.chars().map(|c| {
      if c.is_uppercase() {
          c.to_ascii_lowercase()
      } else {
          c.to_ascii_uppercase()
      }
  }).collect();
  ```
- クロージャの中で `if` 式が値を返している点に注目（`;` を付けない）
