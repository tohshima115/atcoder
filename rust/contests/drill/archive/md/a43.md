# A-43: 加算クエリ

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ があり、最初は全て $0$ である。  
$Q$ 個の操作が与えられる。$i$ 番目の操作は整数の組 $(X_i, V_i)$ であり、「$A_{X_i}$ に $V_i$ を加える」ことを意味する。  
全ての操作を行った後の数列を空白区切りで出力せよ。

### 制約

- $1 \le N \le 100$
- $1 \le Q \le 100$
- $1 \le X_i \le N$
- $0 \le V_i \le 1000$
- 入力は全て整数

### 入力

```
N Q
X_1 V_1
X_2 V_2
...
X_Q V_Q
```

### 出力

操作後の数列を空白区切りで1行に出力せよ。

---

### 入力例1

```
3 2
1 5
3 2
```

### 出力例1

```
5 0 2
```

$A_1$ に $5$、$A_3$ に $2$ を加算 → $(5, 0, 2)$。

---

### 入力例2

```
4 3
2 10
2 5
4 1
```

### 出力例2

```
0 15 0 1
```

$A_2$ に $10$ と $5$ → $15$。

---

### 入力例3

```
1 1
1 7
```

### 出力例3

```
7
```

---

### ヒント

練習ポイント: **クエリのループ処理** と **0初期化の配列**

- $0$ で初期化した配列：
  ```rust
  let mut a = vec![0i64; n];
  ```
- proconio でクエリをまとめて受け取る：
  ```rust
  input! {
      n: usize, q: usize,
      queries: [(usize, i64); q],
  }
  for (x, v) in queries {
      a[x - 1] += v;   // X は 1-indexed
  }
  ```
- `vec![0; n]` は「$0$ が $n$ 個の配列」を作る記法
