# A-45: 切り上げの割り算

配点: 100点

### 問題文

$N$ 個の品物を、1箱に最大 $K$ 個ずつ詰めていく。  
全ての品物を詰めるのに必要な箱の **最小個数** を出力せよ。

### 制約

- $1 \le N \le 10^9$
- $1 \le K \le 10^9$
- 入力は整数

### 入力

```
N K
```

### 出力

必要な箱の最小個数を出力せよ。

---

### 入力例1

```
10 3
```

### 出力例1

```
4
```

$3 + 3 + 3 + 1$ で4箱必要（最後の箱は1個だけ）。

---

### 入力例2

```
9 3
```

### 出力例2

```
3
```

ちょうど割り切れる。

---

### 入力例3

```
1 5
```

### 出力例3

```
1
```

---

### ヒント

練習ポイント: **切り上げ除算**

- 整数の `/` は切り捨てなので、そのままでは「あと1箱」が抜ける
- 切り上げの定番イディオム：
  ```rust
  let ans = (n + k - 1) / k;
  ```
  「割る数より1小さい数を足してから割る」と切り上げになる
- 別解（新しめのRust）: `n.div_ceil(k)` が使える環境ならこれ一発
- 制約が $10^9$ なので `i64` で受ける（`n + k - 1` が `i32` だと溢れる可能性）
