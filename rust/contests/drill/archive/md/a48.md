# A-48: 1回だけ出る値の個数

配点: 100点

### 問題文

$N$ 個の整数 $A_1, A_2, \dots, A_N$ が与えられる。  
数列の中に **ちょうど1回だけ出現する値** が何個あるかを出力せよ。

### 制約

- $1 \le N \le 100$
- $0 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

ちょうど1回だけ出現する値の個数を出力せよ。

---

### 入力例1

```
6
1 2 2 3 3 3
```

### 出力例1

```
1
```

出現回数は `1→1回`, `2→2回`, `3→3回`。ちょうど1回なのは `1` だけ → 1個。

---

### 入力例2

```
3
1 2 3
```

### 出力例2

```
3
```

全部1回ずつ → 3個。

---

### 入力例3

```
2
5 5
```

### 出力例3

```
0
```

`5` は2回出るので該当なし → 0個。

---

### ヒント

練習ポイント: **`HashMap` で出現回数を数える**

- `HashMap<値, 回数>` で各値が何回出たかを集計：
  ```rust
  use std::collections::HashMap;

  let mut count: HashMap<i64, i64> = HashMap::new();
  for &x in &a {
      *count.entry(x).or_insert(0) += 1;
  }
  ```
- `entry(x).or_insert(0)` は「キー x がなければ 0 を入れて、その場所への参照を返す」。`*...  += 1` で回数を増やす
- 集計後、値（回数）が1のものを数える：
  ```rust
  let ans = count.values().filter(|&&v| v == 1).count();
  ```
- `HashMap` の数え上げ（`*entry().or_insert(0) += 1`）は B問題で超頻出のイディオム
