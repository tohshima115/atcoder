# A-49: 数列の合計と最大値

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ が与えられる。
$A$ の **合計** と **最大値** をそれぞれ出力せよ（2行）。

### 制約

- $1 \le N \le 100$
- $-10^9 \le A_i \le 10^9$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

1行目に合計、2行目に最大値を出力せよ。

---

### 入力例1

```
5
1 2 3 4 5
```

### 出力例1

```
15
5
```

---

### 入力例2

```
3
-1 -5 -3
```

### 出力例2

```
-9
-1
```

負の数だけでも最大値は存在する（一番大きい＝0に近い方）。

---

### 入力例3

```
1
42
```

### 出力例3

```
42
42
```

---

### ヒント

練習ポイント: **`iter().sum()` と `iter().max()`** の定番

- 合計:
  ```rust
  let sum: i64 = a.iter().sum();
  ```
  型注釈（`: i64`）が必須。`sum()` は集める型を推論する必要がある
- 最大値:
  ```rust
  let mx = *a.iter().max().unwrap();
  ```
  `max()` は `Option<&T>` を返すので `unwrap()` + `*` でデリファレンス
- 制約に $10^9$ が出てきたら **オーバーフロー注意**: 合計が $N \times 10^9 = 10^{11}$ になりうるので `i64`
