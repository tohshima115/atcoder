# A-5: 文字数を数えよ

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。  
$S$ の長さ（文字数）を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

$S$ の文字数を整数で出力せよ。

---

### 入力例1

```
hello
```

### 出力例1

```
5
```

---

### 入力例2

```
a
```

### 出力例2

```
1
```

---

### 入力例3

```
atcoderbeginnercontest
```

### 出力例3

```
22
```

---

### ヒント

練習ポイント: `String` の長さは `.len()` で取れる

```rust
let n = s.len();
```

**注意**: `.len()` は **バイト数** を返す。英小文字（ASCII）なら文字数と一致するが、日本語などのマルチバイト文字を含むと異なる。文字数が欲しいときは `s.chars().count()` が安全。
