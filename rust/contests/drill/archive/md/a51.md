# A-51: 配列の左ローテート

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ と整数 $K$ が与えられる。
$A$ を **左に $K$ 個ローテート** した数列を空白区切りで出力せよ。

左ローテートとは、先頭の要素を末尾に移す操作を $K$ 回繰り返すこと。

### 制約

- $1 \le N \le 100$
- $0 \le K \le N$
- $0 \le A_i \le 1000$
- 入力は全て整数

### 入力

```
N K
A_1 A_2 ... A_N
```

### 出力

ローテート後の数列を空白区切りで1行に出力せよ。

---

### 入力例1

```
5 2
1 2 3 4 5
```

### 出力例1

```
3 4 5 1 2
```

左に2個ローテート → `(3, 4, 5, 1, 2)`。

---

### 入力例2

```
4 0
10 20 30 40
```

### 出力例2

```
10 20 30 40
```

$K=0$ は変化なし。

---

### 入力例3

```
3 3
1 2 3
```

### 出力例3

```
1 2 3
```

$K=N$ は元に戻る。

---

### ヒント

練習ポイント: **`.rotate_left(k)`** メソッド

- `Vec` には専用メソッドがある：
  ```rust
  a.rotate_left(k);
  ```
- `mut a` で受けてその場で書き換える
- `rotate_right(k)` もある（右ローテート）
- 自力実装するなら:
  ```rust
  let b: Vec<usize> = a[k..].iter().chain(a[..k].iter()).copied().collect();
  ```
  `chain` は2つのイテレータを連結する
