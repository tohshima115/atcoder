# A-53: 数字文字列の各桁の和

配点: 100点

### 問題文

数字のみからなる文字列 $S$ が与えられる。$S$ の **各桁の数字の和** を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は `0`〜`9` のみからなる

### 入力

```
S
```

### 出力

各桁の和を出力せよ。

---

### 入力例1

```
123
```

### 出力例1

```
6
```

$1 + 2 + 3 = 6$。

---

### 入力例2

```
9999
```

### 出力例2

```
36
```

---

### 入力例3

```
0
```

### 出力例3

```
0
```

---

### ヒント

練習ポイント: **`char` → 数値変換**

- `char` を数値に変換する方法は2つ：
  ```rust
  let d: u32 = c.to_digit(10).unwrap();   // 10進数として変換、Option を返す
  let d: i64 = (c as u8 - b'0') as i64;   // ASCIIコードの差で計算
  ```
  - `to_digit(10)` は 0〜9 と a〜z 等を扱える汎用版
  - `c as u8 - b'0'` は ASCII の `'0'=0x30, '1'=0x31, ...` を利用する小ワザ
- 合計の出し方:
  ```rust
  let sum: u32 = s.chars().map(|c| c.to_digit(10).unwrap()).sum();
  println!("{}", sum);
  ```
- `b'0'` はリテラルでバイト値 0x30 を意味する。`'0' as u8` と同じ
