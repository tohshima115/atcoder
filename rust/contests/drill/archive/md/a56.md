# A-56: 降順ソート

配点: 100点

### 問題文

長さ $N$ の数列 $A = (A_1, A_2, \dots, A_N)$ が与えられる。
$A$ を **降順（大きい順）** に並べ替えた数列を空白区切りで出力せよ。

### 制約

- $1 \le N \le 100$
- $0 \le A_i \le 10^9$
- 入力は全て整数

### 入力

```
N
A_1 A_2 ... A_N
```

### 出力

降順に並べた数列を空白区切りで1行に出力せよ。

---

### 入力例1

```
5
3 1 4 1 5
```

### 出力例1

```
5 4 3 1 1
```

---

### 入力例2

```
3
100 200 50
```

### 出力例2

```
200 100 50
```

---

### 入力例3

```
1
42
```

### 出力例3

```
42
```

---

### ヒント

練習ポイント: **降順ソートの3パターン**

3通り書ける。好みでOK:

**① `sort_by` + `cmp` を逆向きに**
```rust
a.sort_by(|x, y| y.cmp(x));   // y と x を逆にして比較
```

**② `sort` → `reverse`**
```rust
a.sort();
a.reverse();
```

**③ `sort_by_key` + マイナス**
```rust
a.sort_by_key(|&x| std::cmp::Reverse(x));   // Reverse でラップ
```

`sort_by` が一番汎用（複雑な比較関数を書ける）。`sort` + `reverse` は脳に優しい。
- 競プロでは ② が単純で読みやすい
- 複合キー（先頭はA昇順、同点ならB降順）になると `sort_by` か `sort_by_key` 必須
