# A-6: 先頭の文字

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。  
$S$ の **先頭の1文字** を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

$S$ の先頭1文字を出力せよ。

---

### 入力例1

```
hello
```

### 出力例1

```
h
```

---

### 入力例2

```
atcoder
```

### 出力例2

```
a
```

---

### 入力例3

```
z
```

### 出力例3

```
z
```

---

### ヒント

練習ポイント: 文字列の先頭1文字の取り出しは複数のやり方がある

```rust
// 方法1: chars() で最初の文字を取り出す
let c = s.chars().next().unwrap(); // char 型

// 方法2: スライスで先頭1バイトだけ取る（ASCII 限定）
let head = &s[..1]; // &str 型
```

`println!("{}", c)` でも `println!("{}", head)` でもどちらでも1文字が出力される。
