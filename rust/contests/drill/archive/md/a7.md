# A-7: 末尾の文字

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。  
$S$ の **末尾の1文字** を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

$S$ の末尾1文字を出力せよ。

---

### 入力例1

```
hello
```

### 出力例1

```
o
```

---

### 入力例2

```
atcoder
```

### 出力例2

```
r
```

---

### 入力例3

```
a
```

### 出力例3

```
a
```

長さ1なら先頭=末尾。

---

### ヒント

練習ポイント: 末尾1文字の取り出し

```rust
// 方法1: chars().last()
let c = s.chars().last().unwrap();

// 方法2: スライス（ASCII 限定）
let tail = &s[s.len() - 1..];
```

`s.len() - 1` を計算するときは `s.len() == 0` ではないことを確認すること（今回の制約では `|S| >= 1` なので安全）。
