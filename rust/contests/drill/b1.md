# B-1: 電球トグル

配点: 200点

### 問題文

$N$ 個の電球があり、最初は全て**消灯**している。  
$Q$ 回の操作があり、$i$ 回目の操作では電球 $T_i$（1-indexed）の状態をトグルする。

- 電球が消灯している場合、点灯させる。
- 電球が点灯している場合、消灯させる。

全ての操作が終わった後、点灯している電球の個数を出力せよ。

### 制約

- $1 \le N, Q \le 1000$
- $1 \le T_i \le N$
- 入力は全て整数

### 入力

```
N Q
T_1 T_2 ... T_Q
```

### 出力

点灯している電球の個数を出力せよ。

---

### 入力例1

```
5 4
1 3 1 5
```

### 出力例1

```
2
```

- 1番: 点灯 → 消灯（2回トグル）
- 3番: 点灯
- 5番: 点灯

3番と5番が点灯 → 2個

---

### 入力例2

```
3 3
2 2 2
```

### 出力例2

```
1
```

2番を3回トグル → 点灯（奇数回）

---

### 入力例3

```
10 1
7
```

### 出力例3

```
1
```

---
