# B-10: 約数の個数

配点: 200点

### 問題文

正の整数 $N$ が与えられる。$N$ の正の約数の個数を出力せよ。

### 制約

- $1 \le N \le 10^{12}$
- $N$ は整数

### 入力

```
N
```

### 出力

$N$ の約数の個数を出力せよ。

---

### 入力例1

```
12
```

### 出力例1

```
6
```

$12$ の約数は $1, 2, 3, 4, 6, 12$ の 6 個。

---

### 入力例2

```
1
```

### 出力例2

```
1
```

---

### 入力例3

```
36
```

### 出力例3

```
9
```

$1, 2, 3, 4, 6, 9, 12, 18, 36$ の 9 個。

---

### 練習ポイント

$1$ から $N$ まで試すと $N \le 10^{12}$ で間に合わない。$\sqrt{N}$ まで調べ、約数 $i$ が見つかったら $N/i$ も約数（$i = N/i$ の重複に注意）。型は $N$ が大きいので注意。
