# B-11: 詰め込める品物の最大数

配点: 200点

### 問題文

$N$ 個の品物があり、$i$ 番目の品物の重さは $w_i$ である。  
容量 $W$ のバッグに、重さの合計が $W$ を超えないように品物を詰めたい。  
詰められる品物の **個数の最大値** を出力せよ。

### 制約

- $1 \le N \le 2 \times 10^5$
- $1 \le W \le 10^{18}$
- $1 \le w_i \le 10^9$
- 入力は全て整数

### 入力

```
N W
w_1 w_2 ... w_N
```

### 出力

詰められる品物の個数の最大値を出力せよ。

---

### 入力例1

```
5 10
3 1 4 1 5
```

### 出力例1

```
4
```

軽い順に $1, 1, 3, 4$ を詰めると合計 $9 \le 10$ で 4 個。次の $5$ を足すと $14 > 10$ なので不可。

---

### 入力例2

```
3 2
5 5 5
```

### 出力例2

```
0
```

---

### 入力例3

```
4 100
10 20 30 40
```

### 出力例3

```
4
```

---

### 練習ポイント

「個数を最大化」したいなら **軽い品物から** 貪欲に詰めるのが最適。ソートしてから合計が $W$ を超えるまで足す。合計が大きくなるので型に注意。
