# B-12: 最頻値

配点: 200点

### 問題文

$N$ 個の整数 $a_1, a_2, \ldots, a_N$ が与えられる。  
最も多く出現する値を出力せよ。そのような値が複数ある場合は、**最も小さい値** を出力せよ。

### 制約

- $1 \le N \le 2 \times 10^5$
- $1 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N
a_1 a_2 ... a_N
```

### 出力

最頻値（同数なら最小の値）を出力せよ。

---

### 入力例1

```
5
1 2 2 3 3
```

### 出力例1

```
2
```

$2$ と $3$ がともに 2 回で最多。小さい方の $2$ を出力。

---

### 入力例2

```
3
7 7 7
```

### 出力例2

```
7
```

---

### 入力例3

```
4
4 3 2 1
```

### 出力例3

```
1
```

すべて 1 回ずつなので、最小の $1$。

---

### 練習ポイント

$a_i$ が最大 $10^9$ なので、出現回数の配列を添字で持つのは無理。**`HashMap` で集計**する。最大回数を求め、その回数を持つキーのうち最小を選ぶ。
