# B-13: 部分集合の和

配点: 200点

### 問題文

$N$ 個の整数 $a_1, a_2, \ldots, a_N$ が与えられる。  
これらの中からいくつか（1 個以上）を選び、選んだ数の合計をちょうど $K$ にできるか判定せよ。  
できるなら `Yes`、できないなら `No` を出力せよ。

### 制約

- $1 \le N \le 20$
- $1 \le K \le 10^9$
- $1 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N K
a_1 a_2 ... a_N
```

### 出力

合計をちょうど $K$ にできるなら `Yes`、できないなら `No` を出力せよ。

---

### 入力例1

```
4 7
1 2 3 4
```

### 出力例1

```
Yes
```

$3 + 4 = 7$ にできる。

---

### 入力例2

```
3 11
1 2 3
```

### 出力例2

```
No
```

全部選んでも $6$ にしかならない。

---

### 入力例3

```
2 2
2 2
```

### 出力例3

```
Yes
```

片方の $2$ を選べばよい。

---

### 練習ポイント

$N \le 20$ なので、$2^N$ 通りの選び方を全部試せる（**bit 全探索**）。各 $0 \le \text{mask} < 2^N$ について、立っているビットの位置の数を足す。合計の型に注意。
