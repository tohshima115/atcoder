# B-14: 最頻値をすべて

配点: 300点

### 問題文

$N$ 個の整数 $a_1, a_2, \ldots, a_N$ が与えられる。  
最も多く出現する値を **すべて**、**小さい順** に出力せよ。

### 制約

- $1 \le N \le 2 \times 10^5$
- $1 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N
a_1 a_2 ... a_N
```

### 出力

最頻値をすべて小さい順に、空白区切りで 1 行に出力せよ。

---

### 入力例1

```
6
1 2 2 3 3 4
```

### 出力例1

```
2 3
```

$2$ と $3$ がともに 2 回で最多。小さい順に並べて出力。

---

### 入力例2

```
3
5 5 5
```

### 出力例2

```
5
```

---

### 入力例3

```
4
4 3 2 1
```

### 出力例3

```
1 2 3 4
```

すべて 1 回ずつなので全部が最頻値。

---

### 練習ポイント

[[drill-b12]] は「最頻値を 1 つ」だったので sort+dedup で逃げられた。今回は **複数答えがある**ので、`HashMap` で出現回数を集計 → 最大回数を求める → その回数を持つキーを集めて sort、が素直。`HashMap` のキーを `Vec` に集めて並べる練習。
