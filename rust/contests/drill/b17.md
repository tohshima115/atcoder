# B-17: 値ごとの個数（昇順）

配点: 300点

### 問題文

$N$ 個の整数 $a_1, a_2, \ldots, a_N$ が与えられる。  
登場する値それぞれについて「値 と その出現回数」を、**値の小さい順** に出力せよ。

### 制約

- $1 \le N \le 2 \times 10^5$
- $1 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N
a_1 a_2 ... a_N
```

### 出力

登場する値ごとに `値 個数` を空白区切りで、値の昇順に 1 行ずつ出力せよ。

---

### 入力例1

```
6
3 1 2 3 1 3
```

### 出力例1

```
1 2
2 1
3 3
```

$1$ が 2 回、$2$ が 1 回、$3$ が 3 回。値の昇順で出力。

---

### 入力例2

```
1
100
```

### 出力例2

```
100 1
```

---

### 入力例3

```
4
5 5 5 5
```

### 出力例3

```
5 4
```

---

### 練習ポイント

出現回数の集計は `HashMap`/`BTreeMap` のどちらでもできるが、**`BTreeMap` ならキーが自動で昇順**なので、集計してそのまま `for (k, v) in &m` で回すだけで答えになる。`HashMap` だと別途キーを集めて sort が要る。「集計 ＋ 昇順出力」は BTreeMap が一番ラク（→ [[../../notes/reference/map-when-and-btreemap]]）。
