# B-18: 最高得点者

配点: 300点

### 問題文

$N$ 人の生徒がいる。$i$ 番目の生徒の名前は文字列 $s_i$、得点は整数 $p_i$ である。  
得点が最も高い生徒の名前を出力せよ。最高得点の生徒が複数いる場合は、**名前が辞書順で最小** のものを出力せよ。

### 制約

- $1 \le N \le 2 \times 10^5$
- $s_i$ は英小文字のみからなる長さ $1$ 以上 $10$ 以下の文字列
- $0 \le p_i \le 10^9$
- 入力は全て整数（$p_i$）と文字列（$s_i$）

### 入力

```
N
s_1 p_1
s_2 p_2
   :
s_N p_N
```

### 出力

条件を満たす生徒の名前を出力せよ。

---

### 入力例1

```
3
alice 80
bob 90
carol 85
```

### 出力例1

```
bob
```

最高得点は 90 の bob。

---

### 入力例2

```
2
zzz 50
aaa 50
```

### 出力例2

```
aaa
```

同点なので辞書順で小さい `aaa`。

---

### 入力例3

```
1
solo 0
```

### 出力例3

```
solo
```

---
