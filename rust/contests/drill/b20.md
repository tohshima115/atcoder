# B-20: 差が K 以下の最長区間

配点: 300点

### 問題文

$N$ 個の整数 $a_1, \ldots, a_N$ が与えられる。  
連続する部分列のうち、（その区間の最大値 $-$ 最小値）が $K$ 以下であるものの中で、**最も長いものの長さ**を求めよ。

### 制約

- $1 \le N \le 3000$
- $1 \le a_i \le 10^9$
- $0 \le K \le 10^9$
- 入力は全て整数

### 入力

```
N K
a_1 a_2 ... a_N
```

### 出力

条件を満たす最長区間の長さを出力せよ。

---

### 入力例1

```
5 2
1 3 2 5 4
```

### 出力例1

```
3
```

`[1,3,2]` は max3 − min1 = 2 ≤ 2 で長さ 3。これより長いものは無い。

---

### 入力例2

```
1 0
5
```

### 出力例2

```
1
```

---

### 入力例3

```
4 100
1 50 2 49
```

### 出力例3

```
4
```

全体で max50 − min1 = 49 ≤ 100。

---

### 練習ポイント（C 頻出：O(N²) 全探索の見抜き）

「区間」と聞くと難しい手法を考えがちだが、**制約が $N \le 3000$** に注目。
全区間は $O(N^2) = 9 \times 10^6$ 通りで、これは間に合う（→ [[../../notes/reference/complexity-estimation|計算量の壁]]）。

各左端 $l$ を固定し、右端 $r$ を伸ばしながら **max/min を $O(1)$ で更新**すれば全区間を $O(N^2)$ で調べられる。
ボーナス：$r$ を伸ばすと max は増え min は減るので **max−min は単調非減少** → $K$ を超えたら `break` してよい。
「制約が小さい → 素直に全探索でいい」と見抜くのが C のひとつの型。
