# B-21: 隣り合わないように選ぶ最大和

配点: 350点

### 問題文

$N$ 個の整数 $a_1, \ldots, a_N$ が一列に並んでいる。  
このうちいくつか（0 個でもよい）を選ぶ。ただし**隣り合う 2 つを同時に選んではいけない**。  
選んだ数の和の最大値を求めよ。

### 制約

- $1 \le N \le 2 \times 10^5$
- $0 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N
a_1 a_2 ... a_N
```

### 出力

隣り合わないように選んだときの和の最大値を出力せよ。

---

### 入力例1

```
5
3 2 5 10 7
```

### 出力例1

```
15
```

$a_1, a_3, a_5 = 3 + 5 + 7 = 15$（隣り合っていない）。

---

### 入力例2

```
1
5
```

### 出力例2

```
5
```

---

### 入力例3

```
4
5 5 5 5
```

### 出力例3

```
10
```

1 つ飛ばしで 2 つ選んで $5 + 5 = 10$。

---

### 練習ポイント（C 頻出：いちばん基本の DP）

貪欲（大きい順に取る）では正しく解けない（取ると隣が取れなくなる影響を考える必要がある）。
これは **DP の最も基本的な形**：

$dp[i]$ = 「$i$ 番目までを見たときの最大和」とすると、$i$ 番目を
- **選ばない** → $dp[i-1]$
- **選ぶ** → $dp[i-2] + a_i$（1 つ前は選べない）

の大きい方：$dp[i] = \max(dp[i-1],\ dp[i-2] + a_i)$。

「直前の状態から次を決める」漸化式を立てるのが DP の入口。和が大きくなるので**型は i64**。
