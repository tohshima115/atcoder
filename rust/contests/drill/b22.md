# B-22: 順位づけ（座標圧縮）

配点: 300点

### 問題文

$N$ 個の整数 $a_1, \ldots, a_N$ が与えられる。  
各 $a_i$ について、それが（重複を除いた）値の中で**小さい方から何番目か**を出力せよ。  
同じ値は同じ順位とする。順位は 1 から始める。

### 制約

- $1 \le N \le 2 \times 10^5$
- $1 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N
a_1 a_2 ... a_N
```

### 出力

$a_1, \ldots, a_N$ それぞれの順位を、空白区切りで 1 行に出力せよ。

---

### 入力例1

```
5
40 10 30 10 50
```

### 出力例1

```
3 1 2 1 4
```

異なる値は $\{10, 30, 40, 50\}$。順位は 10→1, 30→2, 40→3, 50→4。

---

### 入力例2

```
1
100
```

### 出力例2

```
1
```

---

### 入力例3

```
4
5 5 5 5
```

### 出力例3

```
1 1 1 1
```

値が 1 種類なので全部 1 位。

---

### 練習ポイント（C 頻出：座標圧縮）

値が最大 $10^9$ と大きく、そのままでは配列の添字に使えない。**値を「順位」に置き換える**のが座標圧縮。
$10^9$ の値の世界を「1, 2, 3, …」の小さい世界に圧縮することで、後段で配列・BIT などが使えるようになる典型前処理。

手順：①値を `sort` ②`dedup` で重複除去 ③各 $a_i$ が何番目かを `binary_search`（または `BTreeMap`/`HashMap` で値→順位の対応表）で引く。
[[../../notes/reference/map-when-and-btreemap|Mapの使いどころ]]とも繋がる（「値→順位」は Map でも作れる）。
