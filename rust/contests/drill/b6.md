# B-6: 和が K になるペア

配点: 200点

### 問題文

$N$ 個の整数 $a_1, a_2, \ldots, a_N$ が与えられる。  
$1 \le i < j \le N$ を満たす組 $(i, j)$ のうち、$a_i + a_j = K$ となるものの個数を出力せよ。

### 制約

- $2 \le N \le 2000$
- $1 \le K \le 2 \times 10^9$
- $1 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N K
a_1 a_2 ... a_N
```

### 出力

条件を満たす組の個数を出力せよ。

---

### 入力例1

```
5 7
1 2 3 4 5
```

### 出力例1

```
2
```

$(2,5) \to 2+5=7$、$(3,4) \to 3+4=7$ の 2 組。

---

### 入力例2

```
2 10
5 5
```

### 出力例2

```
1
```

---

### 入力例3

```
4 100
1 2 3 4
```

### 出力例3

```
0
```

---

### 練習ポイント

二重ループによる全探索 $O(N^2)$。値の範囲に注意（オーバーフローしない型を選ぶこと）。
