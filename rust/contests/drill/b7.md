# B-7: 隣り合う同じ文字

配点: 200点

### 問題文

英小文字からなる文字列 $S$ が与えられる。  
$S$ の中で **隣り合う 2 文字が同じ** になっている箇所の個数を出力せよ。  
すなわち、$1 \le i \le |S|-1$ で $S_i = S_{i+1}$ となる $i$ の個数である。

### 制約

- $1 \le |S| \le 2 \times 10^5$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

隣り合う同じ文字の箇所の個数を出力せよ。

---

### 入力例1

```
aabbbc
```

### 出力例1

```
3
```

`aa`（1箇所）、`bbb` の中の `bb`（2箇所）で計 3 箇所。

---

### 入力例2

```
abcabc
```

### 出力例2

```
0
```

---

### 入力例3

```
zzzz
```

### 出力例3

```
3
```

---

### 練習ポイント

文字列を隣同士で走査する。`chars()` を `Vec<char>` にするか、`as_bytes()` で添字アクセスするか、`windows(2)` を使うか。
