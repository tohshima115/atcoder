# B-8: 階段の上り下り

配点: 200点

### 問題文

$N$ 段の階段がある。一番下を $0$ 段目、一番上を $N$ 段目とする。  
あなたは最初 $0$ 段目におり、$M$ 個の操作を順に行う。各操作は次のいずれかである。

- `U x` … $x$ 段上る。ただし $N$ 段目を超える場合は $N$ 段目で止まる。
- `D x` … $x$ 段下る。ただし $0$ 段目より下に行く場合は $0$ 段目で止まる。

すべての操作を行ったあと、何段目にいるかを出力せよ。

### 制約

- $1 \le N \le 10^9$
- $1 \le M \le 2 \times 10^5$
- 各操作の $x$ は $1 \le x \le 10^9$ の整数
- 操作の種類は `U` または `D`

### 入力

```
N M
op_1 x_1
op_2 x_2
   :
op_M x_M
```

### 出力

最終的に何段目にいるかを出力せよ。

---

### 入力例1

```
10 3
U 5
U 7
D 3
```

### 出力例1

```
7
```

$0 \to 5 \to \min(12,10)=10 \to 10-3=7$。

---

### 入力例2

```
5 2
D 3
U 1
```

### 出力例2

```
1
```

$0 \to \max(-3,0)=0 \to 1$。

---

### 入力例3

```
100 1
U 50
```

### 出力例3

```
50
```

---

### 練習ポイント

操作を素直にシミュレーション。`String` の op と整数 $x$ を 1 行で読む方法、`min`/`max` でのクランプ。
