# B-9: 区間の合計

配点: 200点

### 問題文

$N$ 個の整数 $a_1, a_2, \ldots, a_N$ が与えられる。  
続いて $Q$ 個の質問が与えられる。$i$ 番目の質問は整数 $l_i, r_i$ からなり、  
$a_{l_i} + a_{l_i+1} + \cdots + a_{r_i}$（区間 $[l_i, r_i]$ の合計）を求めよ、というものである。

各質問に答えを出力せよ。

### 制約

- $1 \le N \le 2 \times 10^5$
- $1 \le Q \le 2 \times 10^5$
- $1 \le l_i \le r_i \le N$
- $1 \le a_i \le 10^9$
- 入力は全て整数

### 入力

```
N Q
a_1 a_2 ... a_N
l_1 r_1
l_2 r_2
   :
l_Q r_Q
```

### 出力

各質問の答えを $Q$ 行で出力せよ。

---

### 入力例1

```
5 2
1 2 3 4 5
2 4
1 5
```

### 出力例1

```
9
15
```

$[2,4] = 2+3+4 = 9$、$[1,5] = 1+2+3+4+5 = 15$。

---

### 入力例2

```
3 1
10 20 30
1 1
```

### 出力例2

```
10
```

---

### 入力例3

```
4 2
1 1 1 1
1 4
2 3
```

### 出力例3

```
4
2
```

---

### 練習ポイント

各質問でループすると $O(NQ)$ で間に合わない。**累積和**を前計算して各質問 $O(1)$ で答える。1-indexed と累積和配列の添字に注意。
