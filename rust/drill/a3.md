# A-3: 最低気温

配点: 100点

### 問題文

$N$ 日間の気温（整数）が与えられる。最低気温を出力せよ。

### 制約

- $1 \le N \le 100$
- $-50 \le T_i \le 50$
- 入力は全て整数

### 入力

```
N
T_1 T_2 ... T_N
```

### 出力

最低気温を整数で出力せよ。

---

### 入力例1

```
5
3 -2 8 1 -5
```

### 出力例1

```
-5
```

---

### 入力例2

```
3
0 0 0
```

### 出力例2

```
0
```

---

### 入力例3

```
1
-50
```

### 出力例3

```
-50
```

---

### ヒント

練習ポイント: `[i64; n]` で配列を読んで最小値を求める

```rust
input! {
    n: usize,
    t: [i64; n],
}
let ans = t.iter().min().unwrap();
```
