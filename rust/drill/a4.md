# A-4: 母音カウント

配点: 100点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。  
$S$ に含まれる母音（a, e, i, o, u）の個数を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

母音の個数を整数で出力せよ。

---

### 入力例1

```
hello
```

### 出力例1

```
2
```

e と o が母音。

---

### 入力例2

```
atcoder
```

### 出力例2

```
3
```

a, o, e が母音。

---

### 入力例3

```
rhythm
```

### 出力例3

```
0
```

母音が1つもない。

---

### ヒント

練習ポイント: `chars()` で1文字ずつ取り出す

```rust
for c in s.chars() {
    match c {
        'a' | 'e' | 'i' | 'o' | 'u' => { /* 母音 */ }
        _ => {}
    }
}
```
