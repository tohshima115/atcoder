# B-2: 人気投票

配点: 200点

### 問題文

$N$ 人がそれぞれ好きな食べ物を1つ回答した。  
最も多く票を得た食べ物の**得票数**を出力せよ。

### 制約

- $1 \le N \le 100$
- 各食べ物の名前は英小文字のみからなり、長さは1以上10以下
- 入力は全て整数または文字列

### 入力

```
N
s_1 s_2 ... s_N
```

### 出力

最大得票数を整数で出力せよ。

---

### 入力例1

```
6
sushi pizza sushi ramen pizza sushi
```

### 出力例1

```
3
```

sushi が3票で最多。

---

### 入力例2

```
4
ramen ramen soba soba
```

### 出力例2

```
2
```

ramen と soba が同数。どちらも2票なので2を出力。

---

### 入力例3

```
1
curry
```

### 出力例3

```
1
```

---

### ヒント

練習ポイント: `HashMap` でカウント

```rust
use std::collections::HashMap;

let mut map: HashMap<String, i64> = HashMap::new();
for food in &foods {
    *map.entry(food.clone()).or_insert(0) += 1;
}
let ans = map.values().max().unwrap();
```
