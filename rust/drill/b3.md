# B-3: K番目の点数

配点: 200点

### 問題文

$N$ 人のテストの点数が与えられる。  
点数を**高い順**に並べたとき、$K$ 番目の点数を出力せよ。

### 制約

- $1 \le K \le N \le 100$
- $0 \le$ 点数 $\le 100$
- 入力は全て整数

### 入力

```
N K
p_1 p_2 ... p_N
```

### 出力

高い順に並べたとき $K$ 番目の点数を出力せよ。

---

### 入力例1

```
5 2
80 60 90 70 50
```

### 出力例1

```
80
```

降順: 90, 80, 70, 60, 50 → 2番目は80。

---

### 入力例2

```
3 3
100 100 100
```

### 出力例2

```
100
```

---

### 入力例3

```
4 1
1 2 3 4
```

### 出力例3

```
4
```

1番目 = 最大値。

---
