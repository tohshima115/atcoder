# B-5: 文字列反転

配点: 200点

### 問題文

英小文字のみからなる文字列 $S$ が与えられる。$S$ を反転した文字列を出力せよ。

### 制約

- $1 \le |S| \le 100$
- $S$ は英小文字のみからなる

### 入力

```
S
```

### 出力

$S$ を反転した文字列を出力せよ。

---

### 入力例1

```
hello
```

### 出力例1

```
olleh
```

---

### 入力例2

```
atcoder
```

### 出力例2

```
redocta
```

---

### 入力例3

```
a
```

### 出力例3

```
a
```

---

### ヒント

練習ポイント: `chars()` で文字のイテレータを作り、`rev()` で逆順にして `collect()` で文字列に戻す

```rust
let ans: String = s.chars().rev().collect();
```

`collect::<String>()` の代わりに型注釈 `let ans: String` でも同じ意味。
