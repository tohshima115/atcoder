## 問題

[[../../../contests/drill/a26|問題文]]

N個の整数の平均を小数第2位まで出力する。

## 実装

```rust
use proconio::input;

fn main() {
    input! {
        n: usize,
        a: [i64; n],
    }
    let sum = a.iter().sum::<i64>();
    let ave: f64 = sum as f64 / n as f64;
    println!("{:.2}", ave)
}
```

**最初の失敗**: `num_traits::float` を使おうとした（不要）。整数同士で割って小数が出なかった。`{}` で出力して `70.00` でなく `70` になった。

## 学んだこと

### 数値型のprefix — i / u / f

| prefix | 意味 | 代表 |
|---|---|---|
| `i` | 符号あり整数（負も含む） | `i64` |
| `u` | 符号なし整数（0以上のみ） | `usize`, `u32` |
| `f` | 浮動小数点 | `f64` |

後ろの数字はビット数。競プロでは基本 `i64` を使い、インデックスは `usize`。

知識レベル: 🔵 説明可能

---

### 整数除算と f64 変換

整数同士で割ると小数が切り捨てられる。平均など小数が必要な場合は `as f64` で変換してから割る。

```rust
let ave = sum / n as i64      // NG: 整数除算（70.00 → 70）
let ave = sum as f64 / n as f64  // OK: 浮動小数点除算
```

知識レベル: 🟢 実装可能

---

### 小数フォーマット `{:.N}`

```rust
println!("{:.2}", 70.0)   // "70.00"（小数2桁固定）
println!("{:.6}", 2.5)    // "2.500000"（小数6桁）
```

`{:.2}` の `:` 以降がフォーマット指定。`.2` が「小数点以下2桁」の意味。競プロでは `{:.2}` か `{:.10}` をよく使う。

知識レベル: 🟢 実装可能

---

### f64 での累乗・ルート

`i64` では整数乗しかできないが、`f64` にすると負の指数・分数乗・ルートが使える。

```rust
10_f64.powi(-2)   // 10^-2 = 0.01（整数乗、i32を受け取る）
2_f64.powf(0.5)   // 2^0.5 = √2（小数乗）
2_f64.sqrt()      // √2（ルートのショートカット）
8_f64.cbrt()      // ∛8 = 2.0（立方根）
```

「小数が絡む計算は全部 `f64` に変換してから」。

知識レベル: 🟡 雰囲気理解
