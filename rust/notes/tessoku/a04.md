# A04 - Binary Representation 1

[[../../tessoku-book/a04|問題文]]

## 問題の要約

10進数の整数 N（1〜1000）を2進数に変換し、10桁ゼロ埋めで出力する。

## 実装アプローチ

### 手動実装（アルゴリズムを理解しながら）

各ビット位置 i（9〜0）について `(n / 2^i) % 2` を計算して上位桁から順に繋げる。

```rust
use proconio::input;

fn main() {
    input! {
        n: i64,
    }
    let mut ans: String = String::new();
    for i in (0..10).rev() {
        let bit = (n / (1 << i)) % 2;
        ans.push_str(&bit.to_string());
    }
    println!("{}", ans);
}
```

### 標準ライブラリを使う方法（競プロで使える）

```rust
println!("{:010b}", n);
```

- `b` → 2進数フォーマット
- `010` → 10桁・0埋め

## このセッションで学んだこと

### Rust のレンジ（Range）の意味

`a..b` は「a以上b未満」。終端を含まない。

| 書き方 | 中身 |
|--------|------|
| `0..3` | 0, 1, 2 |
| `0..=3` | 0, 1, 2, 3 |

- `0..10` と `0..=9` は同じ意味
- 慣習的に `0..10` の方がよく使われる（個数=10 が直感的）

**知識レベル**: 🟢 実装可能

---

### 逆順イテレータ `.rev()`

レンジを逆順に回したいときは `.rev()` を使う。括弧の位置に注意。

```rust
for i in (0..10).rev() { ... }  // 9, 8, 7, ..., 0
// NG: 0..10.rev()  → 10.rev() を呼ぼうとしてコンパイルエラー
```

**知識レベル**: 🟢 実装可能

---

### 左シフト演算子 `<<`

`1 << i` は `2^i` と同じ。2進数の計算では `.pow()` より自然に使える。

```rust
1_i64 << 3  // = 8 = 2^3
```

**知識レベル**: 🟡 雰囲気理解

---

### String への数値追加

数値をそのまま `push_str` には渡せない。`to_string()` で変換してから参照を渡す。

```rust
ans.push_str(&bit.to_string());
```

**知識レベル**: 🟢 実装可能

---

### フォーマット指定子（競プロで便利）

```rust
println!("{:010b}", n);  // 2進数、10桁、0埋め
println!("{:05}", n);    // 10進数、5桁、0埋め
println!("{:x}", n);     // 16進数（小文字）
```

**知識レベル**: 🟢 実装可能
